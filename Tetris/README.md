This is simple tetris game made in pygame.

Description:
You can play your favourite childhood game now in a simplefied form. It contains all the classic tetrominos in their glory, that can be rotated. Complete a row to gain points.

Controls: 
e - rotate right, q - rotate left, d - move left, a - move right, space - slam the tetronimo to the bottom and esc - quit.

Instalation:
To install python go to python.org/downloads and download the latest version of python 3.13.
You also have to have pygame and numpy by writing in cmd "pip install pygame" and "pip install numpy" or "pip install -r requirements.txt" without the double quotes.

How to run:
Write "python main.py" without the double quotes.
